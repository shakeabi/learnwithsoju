Korean hover dictionary — settings

API keys

Both APIs are free. The same NIKL account can register both. Keys are stored locally in chrome.storage.sync and never sent anywhere except the dictionary servers themselves.

Behaviour

Advanced

Power-user settings. Defaults are sensible — tweak only if you know what you want.

Cache

Lookup results are cached locally so the same word never hits the API twice. Clear it if a definition looks stale or you want to free up storage.
